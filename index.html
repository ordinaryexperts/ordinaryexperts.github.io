Services

As an AWS Standard Consulting Partner, we focus on building automated, cost-effective, highly-available and performant AWS infrastructures.

AWS Infrastructure Automation

Let us help you automate your AWS infrastructure. We have expertise in "infrastructure as code" methodologies. With AWS, our toolset normally includes Terraform, CloudFormation, and a configuration management tool like Chef or Ansible.

DevOps & CICD

We use Git Flow, Semantic Versioning and Jenkins (or CodeBuild) to increase your application team's velocity. We implement CodePipeline and CodeDeploy to continuously deliver your product. We bake AMIs with Packer and use Docker for environment parity.

Serverless Application Development

With AWS services like Lambda and DynamoDB, we build cloud-native applications that are ready to scale. We use these with frameworks like Serverless and Apex to build highly-available services with a minimal cost and maintenance overhead.

AWS Account Cost Optimization

"When it comes to AWS support, Ordinary Experts is the company to contact. They not only completed that task at hand but also spent the time to help educate me on certain aspects of AWS. I’m looking forward to working with them again on my next AWS project."
-- Mike Young, Owner, Third Cousin

A user-facing demo environment had gone out of maintenance and become expensive. The customer wanted to recover it to a usable state and reduce costs. There were dev, stage, and production environments each with a dedicated app server and RDS.

We recovered access to inaccessible hosts by mounting EBS volumes to accessible hosts and inspecting the configuration. We inspected app servers and cataloged services and config that were essential to the demo. Because the environment is only a demo, we reclassified it as non-prod and consolidated all app servers into one and all RDS instances into one. We disabled multi-az feature of the remaining RDS instance. We finally migrated config and data to the consolidated host and tore down unused hosts.
